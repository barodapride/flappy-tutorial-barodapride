Day 1 - Project Setup
1. Download the latest Java SDK (JDK - contains core Java Libraries)
2. Download the Android stand alone SDK (contains Android libraries)
3. Download latest IntelliJ Community Edition. I'm using 14.0.3
    IDE - integrated devopment environment
4. Run the libGDX setup ui to generate your project
5. Open IntelliJ and file-import the project using the default gradle wrapper.
6. Set up the desktop run configuration and make its working dir android/assets
7. Verify you can run the desktop program.
